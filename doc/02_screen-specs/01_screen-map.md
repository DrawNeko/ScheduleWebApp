# 画面構成（最新版）

## 1. ログイン画面（`login.html`）

- ユーザID
- パスワード
- ログインボタン
- 認証失敗時エラーメッセージ表示

## 2. トップ画面（`index.html`）

- ログインユーザ表示
- ログアウトボタン
- 表示グループ選択プルダウン
- デフォルトに設定ボタン
- 表示グループ編集ボタン（`group_manage.html` へ遷移）
- 日付移動ボタン（1日前 / 1日後 / 1週前 / 1週先 / 今週）
- 週間スケジュール表（メンバー行 × 7日列）
- 予定チップ：ダブルクリックで編集画面へ
- 空白セル：ダブルクリックで登録画面へ

## 3. スケジュール登録画面（`schedule_add.html`）

- 登録者選択
- 同報者追加（チップ形式）
- タイトル
- 予定日
- 開始時刻 / 終了時刻
- 定期予約ON/OFF
  - ON時: 期間（終了日）、頻度（DAILY / WEEKLY / BIWEEKLY）
- 場所 / メモ / 色
- 登録ボタン / トップへ戻る

## 4. スケジュール編集画面（`schedule_edit.html`）

- 登録画面と同等の項目を編集可能
- `recurring_rule_id` に応じて定期予約ON/OFFを初期表示
- 定期ルールがある場合は期間・頻度を初期表示
- 削除ボタン
- 削除確認モーダル
  - 削除対象のタイトル / 日時 / 場所を表示
  - 定期予定の場合のみ「定期予約ごと削除」チェック表示（初期OFF）

## 5. スケジュール詳細画面（`schedule_detail.html`）

- タイトル、時刻、登録者、場所、メモ、色を表示
- 編集ボタン
- 削除ボタン

## 6. ユーザ登録画面（`user_register.html`）

- ユーザID、ユーザ名、メール、役職
- 登録ボタン

## 7. グループ編集一覧画面（`group_manage.html`）

- 編集可能なグループ一覧表示
- カラム: 編集/削除、グループタイプ、グループ名、ユーザ名（チップ）
- 最下段の `+` 行から新規作成画面へ遷移

## 8. グループ作成/編集画面（`group_form.html`）

- グループタイプ
- グループ名
- 所属ユーザ選択（複数）
- 保存 / 削除 / キャンセル
