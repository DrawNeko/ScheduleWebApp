# 用語集・ドメイン定義

## 1. スケジュール関連

| 用語 | 意味 | 実装上の対応 |
|---|---|---|
| スケジュール | 1件の予定データ | `schedules` テーブル1レコード |
| 登録者 | 予定の作成責任者 | `created_by` |
| 同報者（参加者） | 予定を共有するユーザ | `user_schedule` の紐付け |
| 予定チップ | 週間表の1予定表示UI | `.event-chip` |

## 2. 定期予約関連

| 用語 | 意味 | 補足 |
|---|---|---|
| 定期予約 | ルールに従って複数予定を生成する仕組み | `recurring_rules` と `schedules` で構成 |
| 頻度 `DAILY` | 平日のみ毎日 | 土日除外 |
| 頻度 `WEEKLY` | 毎週 | 同一曜日 |
| 頻度 `BIWEEKLY` | 隔週 | 2週ごと |
| 同期解除 | 定期予約ON→OFF編集時に単体化 | 以後ルール更新の影響を受けない |

## 3. 表示グループ関連

| 用語 | 意味 |
|---|---|
| Publicグループ | 全ユーザが選択可能な表示グループ |
| Privateグループ | 所有者のみ選択可能な表示グループ |
| デフォルト表示グループ | ログイン時に初期選択されるグループ |

## 4. 削除関連

| 用語 | 意味 |
|---|---|
| 単体削除 | 対象日1件のみ削除（`delete_recurring=0`） |
| 一括削除 | 同一ルール配下を削除（`delete_recurring=1`） |

## 5. 認証関連

| 用語 | 意味 |
|---|---|
| セッション | ログインユーザ状態を保持するサーバ側データ |
| `sid` | セッション識別Cookie名 |
